# cli-ai-search

Command line tool for semantic search.

## Requirements

  * Bash
  * coreutils
  * curl
  * jq
  * awk or gawk
  * findutils
  * ocrmypdf
  * tesseract-ocr
  * pandoc
  * sqlite3
  * inotify-tools
  * ollama

## Supported document formats

  * plain text
  * ms-office documents docx
  * libreoffice/openoffice documents odt
  * PDF
  * HTML
  * RTF
  * EPUB
  * rst
  * org
  * markdown
